Function pivot

  • pivot<TSource>(
        source: TSource[],
        dimension: Dimension<TSource>,
    ): Matrix<TSource>

  • Pivots source data by one Dimension returning a Matrix (read on for two or more dimensions).

    Type Parameters

    • TSource

      The type of the source data.

    Parameters

    • source: TSource[]

      The source data, an array of objects.

    • dimension: Dimension<TSource>

      The Dimension used to pivot the source data by.

    Returns Matrix<TSource>

    Example

    The following code creates a Matrix of Player objects, pivoted by their position property:

    const positions: string[] = ['Goalkeeper', 'Defender', 'Midfielder', 'Forward'];
    const x: Dimension<Player> = positions.map(property('position'));

    const matrix: Matrix<Player> = pivot(squad, x);

    See GitHub for a complete example.

  • pivot<TSource>(
        source: TSource[],
        ...dimensions: Dimension<TSource>[],
    ): Hypercube

  • Pivots source data by any number of dimensions returning a Hypercube.

    Type Parameters

    • TSource

      The type of the source data.

    Parameters

    • source: TSource[]

      The source data, an array of objects.

    • ...dimensions: Dimension<TSource>[]

      The dimensions to pivot the source data by.

    Returns Hypercube

    Because of the arbitory number of dimensions that can be passed to this overload, the shape of the Hypercube cannot be known.

    Remarks

    While it is possible to pass no Dimension arguments into this overload of the pivot function, it will result in an exception being thrown.

    Example

    The following code creates a Hypercube, pivoting the source data by three dimensions (though it can be any number):

    const hypercube: Hypercube = pivot(data, z, y, x);

    See GitHub for a complete example.
